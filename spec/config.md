# 项目运行配置

## 一、运行环境配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| Node.js 版本 | >=18.0.0 | 运行时环境 |
| 模块类型 | ESM | `"type": "module"` |
| 数据存储路径 | `./data/students.json` | JSON 数据文件位置 |
| 数据目录 | `./data/` | 数据文件存放目录 |

## 二、测试配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 测试框架 | Vitest | 单元测试与集成测试框架 |
| 测试文件目录 | `./tests/` | 测试文件存放目录 |
| 测试文件匹配 | `tests/**/*.test.js` | 测试文件匹配规则 |
| 测试运行命令 | `npm test` | 运行全部测试 |
| 测试监听命令 | `npm run test:watch` | 开发时监听模式 |

## 三、命令行交互配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 一级菜单选项 | 1-5 | 录入/查询/统计/查看全部/退出 |
| 成绩范围 | 0-100 | 单科成绩有效范围 |
| 学号格式 | 字符串 | 无长度限制，非空即可 |
| 交互方式 | readline | Node.js 内置命令行交互模块 |
| 提示符 | `> ` | 输入提示符 |
| 退出命令 | 5 | 一级菜单选择5退出 |
| 返回菜单 | 输入空或特定字符 | 操作完成后返回主菜单 |

## 四、参数默认值配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| JSON 缩进 | 2 空格 | 数据文件格式化缩进 |
| 平均分精度 | 1 位小数 | 平均分保留小数位数 |
| 文件编码 | UTF-8 | 数据文件编码格式 |
| 列表宽度 | 统一格式化 | 学生列表展示列宽 |
